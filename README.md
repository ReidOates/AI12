# 🌄 AI12 - Scene Classification using Transfer Learning Xception

An AI-powered web application for natural scene classification using **Transfer Learning Xception** and **Flask**.

---

## 📌 Features

- Upload landscape images
- Predict one of six scene categories
- Display prediction confidence
- Responsive web interface
- Ready for Hugging Face Spaces deployment

---

## 🖼️ Dataset

Intel Image Classification Dataset

Classes:

- Buildings
- Forest
- Glacier
- Mountain
- Sea
- Street

Dataset Source:
https://www.kaggle.com/datasets/puneet6060/intel-image-classification

---

## 🧠 Model

- Transfer Learning Xception
- TensorFlow 2.21
- Fine Tuning
- Input Size 299 × 299

## 🚀 Installation

Clone repository

```bash
git clone https://github.com/ReidOates/AI12.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run application

```bash
python app.py
```

Open

```
http://localhost:7860
```
